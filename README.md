# Victor – AI Trainer Elektrotechniek

Victor is een AI-chatbot voor elektrotechnische e-learning, ontwikkeld voor Veritech. Hij is bedoeld als ondersteuning binnen Rise-modules en Moodle.

## Wat doet Victor?
- Stelt bij de eerste vraag een diagnosevraag om het niveau van de deelnemer in te schatten
- Geeft daarna direct en volledig antwoord, afgestemd op dat niveau
- Vraagt na elk antwoord of het duidelijk was
- Verwijst waar relevant naar NEN 3140 en NEN 3840

## Victor gebruiken in Rise
1. Deploy Victor naar Vercel (zie technische documentatie)
2. Kopieer de Vercel-URL
3. Voeg in Rise een knop toe die de URL opent in een nieuw tabblad

## Victor gebruiken in Moodle
Voeg een HTML-blok toe met:
```html
<iframe src="https://jouw-vercel-url.vercel.app" width="480" height="600" frameborder="0"></iframe>
```

## Nieuw gesprek starten
Klik op de knop "Nieuw gesprek" rechtsboven in het chatvenster.
